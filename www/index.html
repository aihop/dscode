dscode.

AI agent in your pocket · powered by DeepSeek
$ curl -fsSL https://dscode.org/install.sh | sh

Why dscode?

  • Pure CLI — no web UI, no TUI, no bloat
  • Mobile-first — SSH from your phone, Termux, iSH
  • DeepSeek-native — V4 Pro, V4 Flash, R1, V3
  • Single 7.5MB binary — statically linked, ARM-ready
  • Streaming responses — see tokens as they arrive
  • Session persistence — pick up where you left off

Install

# One line (recommended)
$ curl -fsSL https://dscode.org/install.sh | sh

# Or with Cargo
$ cargo install dscode

# Or build from source
$ git clone --recursive https://github.com/aihop/dscode
$ cd dscode && cargo build --release -p dscode

Quickstart

# Set your API key
$ dscode auth login
Enter your DeepSeek API key: sk-...

# Start chatting
$ dscode chat

# One-shot prompt
$ dscode run "write fibonacci in python"

# Use Flash model for speed
$ dscode chat -m flash

Commands

  chat      Interactive chat with DeepSeek
  run       Single prompt, stream response
  auth      Login / logout / status
  config    View or change settings
  session   List, resume, export conversations
  model     List models and capabilities
  tools     Manage tool access

On your phone

# Termux (Android)
$ pkg install curl
$ curl -fsSL https://dscode.org/install.sh | sh
$ dscode auth login
$ dscode chat

# SSH from anywhere
$ ssh user@your-server
$ dscode chat

# iSH (iOS)
$ apk add curl
$ curl -fsSL https://dscode.org/install.sh | sh

Auto-detects narrow terminals. No horizontal scroll on phone screens.
